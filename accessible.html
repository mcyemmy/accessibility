Keyboard Accessibility

This is a demonstration of a Keyboard Accessible Navigation menu for the disabled user. I implemented this using semantic markup, role attributes, aria attributes and tabindex. Tab to the button and press Enter key

This instruction helps you navigate this web app page. To view the navigation menu for this page, tab to the menu icon below, then press Enter on the menu Icon (button) to view the nav list. Tab to the first child on the menu list, then use your keyboard up and down arrow key to move between the nav links.
You can press the Enter key on the nav links to visit their landing page and can return to this app page by pressing the Alt key + left arrow.
Javascript(JQuery) and CSS for this web app are to the left of this panel for you to review. I can further explain this initiative at the interview. Thanks for viewing.
